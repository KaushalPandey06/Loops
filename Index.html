loops

a loop is a control structure that allows a certain block of code to be executed repeatedly, either a specific number of times or until a particular condition is met. Loops help in automating repetitive tasks and are fundamental for efficient and concise program design. There are three loops available in C-language.


  1. while loop
  2. do-while loop
  3. for loop



while loop

a while loop is a control flow statement that allows code to be executed repeatedly as long as a specified condition is true.

**important points :-

  • while loop first checks the condition if the condition is true, the block of code execute. Otherwise, the block of code never gonna execute.

  • minimum execution of while loop is "zero".

  • While loops are suitable when the number of iterations is not known beforehand or based on a dynamic condition.

  • It means if we does not know how many times we need to execute the code, we only know about the specific condition we need to stop, then while loop is used.

general syntax :-


while ( condition )
{

//code to be executed

}

do-while loop

do-while loop is used to repeatedly execute a block of code while a given condition is true.

**important points :-

  • It's function is same as while loop.

  • do-while loop extecute the block of code once, without checking the condition.

  • minimum execution of do-while loop is "one".

  • do-while loops are suitable for that programs in which a specific code is need to run and then depends upon the user to proceed further or not. This is done by set a condition to ask for user's opinion.

general syntax :-

for loop

A for loop is a control flow statement in many programming languages, including C. It provides a concise way to write a loop that repeats a specific block of code for a predetermined number of times. The basic syntax of a for loop includes three components: initialization, condition, and update. These components work together to control the loop's execution.

**important points :-

  • Initialization: This part is executed once at the beginning of the loop and is often used to initialize a loop control variable.

  • Condition: The loop continues executing as long as this condition is true. If the condition becomes false, the loop exits.

  • Update: This part is executed after each iteration of the loop and is typically used to update the loop control variable. Generally Increment and Decrement is used for update.

general syntax :-


for(initialization ; condition ; update)
{

// code to be repeated

}
